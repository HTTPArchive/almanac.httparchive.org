Três anos atrás, me perguntei: muitas ferramentas podem me dizer o quão bem construído está meu site, mas onde eu poderia ir para ver o estado da web como um todo? Por mais sofisticado que seja o conjunto de dados do HTTP Archive, as respostas que ele nos fornece só podem ser úteis como as perguntas que fazemos a ele. Sou desenvolvedor web, mas não sou especialista em todas as áreas do desenvolvimento web - ninguém é esperado que seja! Mas coletivamente, todos nós temos nossas próprias áreas de especialização. Reunindo um número suficiente de pessoas, podemos começar a fazer as perguntas certas sobre o estado da web que o HTTP Archive pode responder de maneiras realmente significativas. Essa foi a ideia original por trás do Web Almanac.

Este ano estamos de volta com a terceira edição, que foi possível graças ao trabalho árduo de mais de cem pessoas incríveis da comunidade da web. Gostaria de chamar especificamente algumas pessoas para quem este é o terceiro ano consecutivo contribuindo: Barry Pollard, David Fox, Paul Calvano, Brian Kardell, Doug Sillars, Eric Portis, Thomas Steiner, Robin Marx, Alan Kent, and Abby Tsai. . Tenho uma enorme dívida de gratidão com todos os colaboradores por doarem seu tempo a este projeto, mas especialmente a essas 10 pessoas que fazem parte dele desde o início.

A edição de 2021 consiste em uma linha abrangente de 24 capítulos, incluindo dois que estamos ansiosos para abordar pela primeira vez: Structured Data and WebAssembly. Esses novos capítulos nos ajudam a expandir o escopo do Web Almanac, o que educa nossa base de leitores sobre uma gama mais diversificada de tópicos e fornece dados acionáveis para grupos ainda mais especializados. Em última análise, é por isso que fazemos isso: esperamos que nossa pesquisa possa ser utilizada pela comunidade web como uma fonte compartilhada de verdade para melhorar significativamente o ecossistema. Se você encontrar este recurso tão valioso quanto nós, adoraríamos se você o compartilhasse com outras pessoas interessadas no estado da web. Juntos, vamos usar esses dados como uma força impulsionadora para a mudança positiva.

— Rick Viscomi, editor-chefe do Web Almanac
