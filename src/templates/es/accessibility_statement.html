Accessibility Statement

Visión general

Web Almanac se compromete a garantizar la accesibilidad digital para todas las personas. Mejoramos continuamente la experiencia del usuario para todos y aplicamos relevantes estándares de accesibilidad.

Esta declaración fue revisada por última vez el 2 de junio de 2020.

Medidas para apoyar la accesibilidad

Web Almanac toma las siguientes medidas para garantizar la accesibilidad de este sitio web:

  • Incluir la accesibilidad como parte del desarrollo de este sitio web.
  • Revisar la accesibilidad utilizando las herramientas disponibles.
  • Ser transparentes sobre los problemas de accesibilidad en nuestra página de GitHub.
  • Abordar los problemas de accesibilidad que se planteen de manera oportuna.
  • Publicar y mantener actualizada esta declaración de accesibilidad.

Estado de conformidad

Las Web Content Accessibility Guidelines (WCAG) definen los lineamientos para que diseñadores y desarrolladores apliquen y mejoren la accesibilidad, especialmente a personas con discapacidades. Define tres niveles de conformidad: Nivel A, Nivel AA y Nivel AAA. El Web Almanac cumple parcialmente con WCAG 2.1 Nivel AAA. Parcialmente significa que algunas partes del contenido no se ajustan completamente al estándar de accesibilidad.

Nos consideramos parcialmente conformes porque no hemos completado una auditoría completa, verificada externamente, del sitio web, ni los usuarios de lectores de pantalla lo han revisado, por lo que no podemos estar seguros de nuestro estado actual. Hemos auditado todas las páginas del sitio web utilizando varias herramientas., incluyendo el WAVE Accessibility Tool, el Firefox Accessibility Inspector y el AXE Chrome plugin.

El propósito de Web Almanac es proporcionar información detallada sobre el estado de la web. Si bien nuestro objetivo es garantizar que todo el contenido sea leído fácilmente, incluso por no expertos, la naturaleza del material puede significar que puede haber un nivel de experiencia y comprensión de la tecnología necesarios para comprender algunos capítulos y algunas métricas, que puede no cumplir con las pautas WCAG 2.1 Nivel AAA de nivel de lectura.

También usamos Google Sheets para algunas visualizaciones interactivas, que también puede no cumplir con los estándares de accesibilidad WCAG. Estos son cargados vía iframes y proporcionamos imágenes para todas nuestras figuras, y solo mostramos visualizaciones interactivas cuando el software cliente lo soporta. Si utiliza un navegador web y no desea ver las visualizaciones interactivas, entonces las imágenes se pueden hacer ver agregando un ?print como parámetro de consulta a cualquiera de los capítulos, vea el capítulo de Accesibilidad en modo de impresión para ver un ejemplo. Podemos buscar que esta funcionalidad sea más fácil de activar si es necesario, pero todas las visualizaciones tienen descripciones de texto detalladas, por lo que no creemos que sea necesario. Por favor haznos saber si estamos en lo cierto o no acerca de esto.

En algunos capítulos, también incluimos otro contenido en el sitio, no creado por nosotros, incluidos videos de YouTube, enlaces PDF o enlaces a otros artículos, que pueden no cumplir con los estrictos criterios requeridos por WCAG 2.1 nivel AAA. Consulte la nota a continuación sobre contenido externo.

La versión PDF del Web Almanac sigue las mejores prácticas de accesibilidad de PDF y cumple con el estándar PDF/UA-1.

Contenido externo

Algunos capítulos enlazan a otro contenido (videos, PDFs, artículos) los cuales no están bajo nuestro control y que pueden no alcanzar los altos niveles de conformidad que hemos logrado. Donde sea posible, proporcionaremos enlaces a artículos en el mismo idioma de la página que se está viendo, pero esto no siempre es posible.

Compatibilidad con navegadores y tecnología de asistencia

Web Almanac está diseñado para ser compatible con tantos navegadores y tecnologías de asistencia como sea posible. El sitio web se ha creado de manera progresiva y es completamente funcional incluso sin admitir todas las características modernas que usemos, pero puede notar un deterioro en las imágenes y ausencia de algunas funciones (imágenes interactivas, recuento de comentarios etc.) cuando no se usa un navegador moderno o una versión reciente de Chrome, Safari, Firefox o Edge.

En este momento, no hemos completado las pruebas en hardware o software de tecnología de asistencia, pero hemos utilizado herramientas, como se mencionó anteriormente, para intentar abordar cualquier problema en estos dispositivos.

Especificaciones técnicas

El sitio web está construido sobre, HTML semántico (incluyendo WAI-ARIA donde corresponda), mejorando con CSS para los estilos y JavaScript limitado para algunos componentes interactivos. El sitio web hace uso de prácticas de desarrollo modernas como CSS Grid, Fetch, y fuentes (tipografía) WOFF2, que no son compatibles con antiguos navegadores como IE 11. El sitio web sigue siendo completamente funcional en estos navegadores, pero puede que no sea tan atractivo visualmente. No se deben requerir complementos o bibliotecas adicionales para ver el sitio web.

El PDF de Web Almanac sigue los mismos estándares de accesibilidad que el sitio web y, además, está etiquetado estructuralmente para facilitar la accesibilidad.

Enfoque de evaluación

Autoevaluación: el equipo de Web Almanac evaluó el contenido utilizando su propia experiencia y una serie de herramientas, incluida la WAVE Accessibility Tool, el Firefox Accessibility Inspector y el AXE Chrome plugin. El PDF de Web Almanac se probó con un escaneo de accesibilidad en Acrobat Reader Pro DC.

Retroalimentación

Agradecemos sus comentarios sobre la accesibilidad de este sitio web. Las mejores maneras de llegar a nosotros sería plantear un problema en GitHub o envíenos un correo electrónico a team@httparchive.org. También agradeceríamos Pull Request para aquellos que deseen contribuir directamente. Web Almanac es un proyecto basado en la comunidad, administrado por voluntarios, por lo que no podemos garantizar un plazo de tiempo para responder a los comentarios, pero responderemos lo antes posible.
